# blog

个人技术博客，域名：https://blog.lishunxing.cn

## 技术栈

### Java

- **Spring 生态**：Spring Framework、Spring Boot、Spring Cloud 微服务解决方案
- **JUC 并发编程**：Java 并发编程核心包，线程池、锁机制、并发集合
- **数据库**：MySQL 索引、事务、锁机制，Redis 数据结构与持久化
- **消息队列**：RabbitMQ 消息中间件，异步通信与解耦

### 软件工具

- Jenkins 自动化部署
- Canal 数据同步

### 数据结构与算法

- LeetCode 刷题笔记
- 递归、数组、链表等基础算法

## 本地运行

```bash
npm install
npm run docs:dev
```

## 构建部署

```bash
npm run docs:build
```

基于 [VitePress](https://vitepress.dev/) 构建
